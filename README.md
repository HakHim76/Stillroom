# Stillroom

A calm, constraint-based focus system:
fewer tasks → deeper focus → reflection → growth.

## Stack

- React + Vite (client)
- Express + MongoDB (server)
- Session auth (cookies)

## Features

- Max 3 prioritized tasks
- One focus session per task
- Reflection locks tasks permanently
- Flash feedback system
- Clean, minimal UI

## Local Setup

### 1 Install

```bash
cd server && npm install
cd ../client && npm install
```
